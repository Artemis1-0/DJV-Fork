Building the Source Code
Required build tools:
  • C++11 compiler
  • CMake 3.12
  • git
Required libraries:
  • GLM 0.9.9
  • IlmBase 2.3
  • OpenColorIO 1.1.1
  • OpenGL 3.3
Optional libraries:
  • FFmpeg 4.1.3
  • OpenEXR 2.3
  • libjpeg-turbo 2.0
  • libpng 1.6
  • libtiff 4.0
The following steps describe how to compile both the DJV source code and the third party libraries that it depends on.
First clone the repository:
> git clone https://github.com/darbyjohnston/DJV.git djv-git
Then create a new sub-directory and start the build:
> mkdir djv-git-Debug
> cd djv-git-Debug
> cmake ../djv-git -DCMAKE_BUILD_TYPE=Debug -DDJV_THIRD_PARTY=TRUE -DCMAKE_FIND_FRAMEWORK="LAST"
> make
After the build completes add the install path to your DYLD_LIBRARY_PATH: 
> export DYLD_LIBRARY_PATH=$DYLD_LIBRARY_PATH:$PWD/install/lib
Run the confidence tests:
> make test
Run the djv application:
> ./build/bin/djv
