Magnify Tool

This tool provides image magnification.

  1. Magnification view
  2. Magnification slider
  3. Set whether the color profile is enabled
  4. Set whether the display profile is enabled

Color Picker Tool

This tool provides color picking.

  1. Color swatch
  2. Color sliders
  3. Pixel information and lock
  4. Color picker sample size
  5. Set whether the color profile is enabled
  6. Set whether the display profile is enabled

Histogram Tool

This tool provides an image histogram.

  1. Histogram view
  2. Minimum pixel value
  3. Maximum pixel value
  4. Histogram size
  5. Set the pixel mask
  6. Set whether the color profile is enabled
  7. Set whether the display profile is enabled

Info Tool

This tool provides image information.

  1. File name
  2. Current file layer
  3. Image size and aspect ratio
  4. Pixel information
  5. Time information
  6. Image tags
