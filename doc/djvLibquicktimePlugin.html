libquicktime

This plugin provides support for libquicktime, an open source library for reading and writing movies.

File extensions: .qt, .mov, .avi, .mp4

Supported features:

  • 8-bit RGBA
  • File compression

Command Line Options

-libquicktime_codec (value)Set the codec used when saving Libquicktime movies. Options = rtjpeg, raw, rawalpha, v308, v408, v410, yuv2, yuv4, yv12, 2vuy, v210, yuvs, jpeg, mjpa, png, pngalpha, ffmpeg_mpg4, ffmpeg_msmpeg4v3, ffmpeg_msmpeg4v3_wmp, ffmpeg_h263, ffmpeg_h263p, ffmpeg_mjpg, ffmpeg_rle, ffmpeg_dv, ffmpeg_dvcpro, ffmpeg_dv50, ffmpeg_ffvhuff, ffmpeg_ffv1, ffmpeg_dnxhd, ffmpeg_imx. Default = mjpa.
