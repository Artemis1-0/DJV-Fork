Change List
Version 2.0.3 Beta
  • Improved monitor DPI detection
  • Add a setting for which monitor is used for full screen mode
  • Improved frame number handling
  • Improved memory cache performance
  • The memory cache is now a global setting instead of per file
  • Add a new action to "frame" the view, so the image fills the window without being obscured by the user interface
  • Add a new action to reset the zoom
  • Add a keyboard shortcut for the color space widget
  • Use the escape key as an alternative to exit full screen mode
  • Disable audio controls when there is no audio
  • Limit the memory cache to the physical memory size
  • Add a new action to fit the window to the image
Version 2.0.2 Beta
  • Improved memory cache performance
  • Improved DPX and Cineon playback performance
  • Fixed 32-bit integer audio playback
  • Added a setting for the number of FFmpeg threads
Version 2.0.1 Beta
  • Improved memory cache performance
  • Added a setting for the number of I/O threads
  • Middle mouse button pans the view
  • Right mouse button scrubs frames
  • Scrubbing frames does not stop playback
  • File sequenceces are enabled by default
  • Fixed upside down TIFF bug
