Overview

Images, image sequences, and movie files may be opened from the "File" menu, by dragging and dropping them onto the image view, or by listing them on the command-line.

A memory cache stores images for fast playback performance, you can change the size in the preferences. You can also disable the cache if you want to stream images directly from disk.

If you don't mind sacrificing image quality you can speed up image loading and reduce memory consumption by using either "proxy scaling" or "8-bit conversion". Proxy scaling reduces the resolution of the images as they are loaded, 8-bit conversion reduces their bit-depth.

Actions

Opening and closing:

Icon Menu Item Description Keyboard Shortcut
Open Open a new file Ctrl O
Reload Reload the current file Ctrl R
Reload Frame Reload the current frame Shift Ctrl R
Close Close the current file Ctrl W

Exporting:

Menu Item Description Keyboard Shortcut
Export Sequence Export the current sequence Ctrl S
Export Frame Export the current frame Shift Ctrl S

Layers:

Menu Item Description Keyboard Shortcut
Next Layer Go to the next layer Ctrl =
Previous Layer Go to the previous layer Ctrl -

Proxy scaling and 8-bit conversion:

Menu Item Description
Proxy Scale / None Disable proxy scaling
Proxy Scale / 1/2 Reduce the resolution of loaded images by 1/2
Proxy Scale / 1/4 Reduce the resolution of loaded images by 1/4
Proxy Scale / 1/8 Reduce the resolution of loaded images by 1/8
8-bit Conversion Toggle whether 8-bit conversion is enabled

Memory cache:

Menu Item Description
Memory Cache Toggle whether the memory cache is enabled
Preload Cache Load frames automatically when playback is stopped
Clear Cache Toggle whether the memory cache is enabled
