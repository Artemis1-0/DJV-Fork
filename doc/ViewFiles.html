File Sequences

A file sequence is a group of files that represent individual frames of a film or animation sequence. DJV will recognize these groups of files and display them in the user interface as a sequence:

Individual FilesFile Sequence
render.1.exr
render.2.exr
render.3.exr 		render.1-3.exr

For a file to be recognized as part of a sequence it must have a frame number at the end of it's name and the same file extension. Here are some examples of different naming conventions for file sequences:

File NameBase NameFrame NumberExtension
render.1.exrrender.1.exr
render.0001.exrrender.0001.exr
render1.exrrender1.exr
render0001.exrrender0001.exr
0001.exr0001.exr

Formatting

File sequences can be formatted in different ways:

OffSparseRange
render.1.exr
render.2.exr
render.3.exr
render.5.exr 		render.1-3,5.exr render.1-5.exr

The "sparse" setting will show the exact frames in the sequence including any gaps where there may be missing frames. The "range" setting will show a continous sequence even if there are missing frames. The formatting can be set either in the file browser or the preferences.

Image File Formats

Image file formats: Cineon, DPX, IFF, JPEG, OpenEXR, PIC, PNG, PPM, RLA, SGI, Targa, TIFF

Movie file formats: MP4, ProRes, MJPEG, IFL

Opening Files

There are multiple ways to open files:

  • Using the file browser
  • Choosing a file from the recent files list
  • Dragging and dropping a file onto the image view
  • Specifying arguments on the command line
