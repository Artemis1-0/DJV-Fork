Opening Files

There are a number of ways to open files:

  • Using the file browser
  • Choosing a file from the recent files list
  • Dragging and dropping a file on the image view
  • Specifying arguments on the command line

Menu Items

Icon Name Description Keyboard Shortcut
Open Show the file browser Ctrl + O
Recent List of recent files
Reload Reload the current file Ctrl + R
Reload Frame Reload the current frame Shift + Ctrl + R
Close Close the current file Ctrl + W
