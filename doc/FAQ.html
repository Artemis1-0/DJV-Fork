    • Is DJV free?

    The DJV source code is covered by an open source BSD style license, which means that it is basically free to use, modify, and distribute, as long as the copyright notices and license are retained. See the legal page for additional information.

    Note that some of the third party software that DJV depends on may have different licensing requirements, see the legal page for additional details.

    How do I setup the DJV command line tools?

    You can find information about setting up the command line tools on the installation page.

    How do I use a LUT (lookup table) with the DJV viewer?

    You can find information about using LUTs on the color and display profiles page.

    Are 3D LUTs supported?

    3D LUTs are not currently supported.

    How can I convert an image sequence to a movie (or vice versa)?

    To convert an image sequence or movie you can use either the djv_view application or the djv_convert command line tool.

    Here is an example of using the djv_convert command line tool:

    > djv_convert input.1-100.sgi output.mp4
