djv_view

The djv_view application provides real-time image sequence and movie playback. It is intended as a professional tool for reviewing computer animation, film, and video footage.

This screenshot shows the main window with the major user interface components:

  1. Menu bar and tool bar
  2. Image view
  3. Playback controls and information
  4. Information about the selected pixel, current image, and state of the memory cache
