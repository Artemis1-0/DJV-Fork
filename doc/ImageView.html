Resizing the View
Full
Set the view so the image fills the window.
Frame
Set the view so the image fills the window without being obscured by the user interface.
Center
Set the view so the image is centered and the zoom is 1:1.
Panning and Zooming
You can pan the view by either activating the pan tool or using the middle mouse button. To zoom in or out use the menu items "View/Zoom In", "View/Zoom Out", or "View/Zoom Reset".
