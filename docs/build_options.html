Build Options

These CMake variables can be used to customize how DJV is built:

Variable Type Default Description
DJV_OPENGL_ES2 Boolean False OpenGL ES 2.0 rendering. Experimental.
DJV_PYTHON Boolean False Python support. Experimental.
DJV_BUILD_BIN Boolean True Build binaries.
DJV_BUILD_DOCS Boolean True Build documentation.
DJV_BUILD_EXAMPLES Boolean True Build examples.
DJV_TINY_BUILD Boolean False Enable a "tiny" build (only core libraries + tests).
DJV_GPROF Boolean False Enable gprof profiling.
DJV_GCOV Boolean False Enable gcov code coverage targets.
DJV_THIRD_PARTY Boolean True Build third party packages.
DJV_THIRD_PARTY_OPTIONAL Boolean True Build optional third party packages.
DJV_THIRD_PARTY_DISABLE_BUILD Boolean False This is used to disable ExternalProject_Add() for third party packages. This can be helpful for speeding up compile iterations after the third party packages have been built.
