Configurations

DJV uses the OpenColorIO library for color management and includes a few of the sample color configurations. Additional configurations, such as the ACES configurations, can be downloaded here.

Color Space Widget

The color space widget allows you to configure color management settings. By default it is configured with a few of the sample configurations from OpenColorIO.

To open the color space widget:

  1. Open the "Image" menu
  2. Click the menu item "Color Space"
  3. The configuration tab
  4. The view tab
  5. The image tab

The "Config" tab allows you to set the active configuration, and add or remove configurations. The "Display" tab allows you to set the color space for the display. The "Image" tab allows you to set the color space used for each file type.

Adding ACES Configurations

Download the sample OCIO configurations, which include the ACES configurations, here. Unpack the download to a location where you want to store the configurations, and use the color space widget to add them:

  1. Open the color space widget and click the add configuration button
  2. Navigate to the directory where you unpacked the download, and click on the directory with the ACES version you want to add.
  3. Click on the config.ocio file to add the configuration.
  4. The configuration is added and made current.

The display color space will automatically be set to the defaults in the configuration. To change the color space click on the color space name:

  1. Go to the display tab
  2. Set the display color space
  3. Set the view color space

Setup color spaces for the different file types:

  1. Go to the image tab
  2. Click the add color space button
  3. Choose a file type from the popup
  4. Click the color space button
  5. Choose a color space from the popup
