Linux

RPM

Installing a RPM package:

sudo rpm -i DJV2-2.0.4-1.x86_64.rpm

The default installation directory is "/usr/local/DJV2".

Starting the DJV viewer:

/usr/local/DJV2/bin/djv.sh

Uninstalling the RPM package:

sudo rpm -e DJV2

DEB

Installing a Debian package:

sudo dpkg -i DJV2_2.0.4_amd64.deb

The default installation directory is "/usr/local/DJV2".

Starting the DJV viewer:

/usr/local/DJV2/bin/djv.sh

Uninstalling the Debian package:

sudo dpkg --remove DJV2

Apple macOS

Standard macOS DMG files are provided for installation. To install, double-click the DMG file and drag and drop the application to where you would like it installed.

Microsoft Windows

Standard Windows MSI files are provided for installation. To install, double-click the MSI file to start the installation process.
