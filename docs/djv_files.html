Opening files

Open the file browser:

  1. Open the "File" menu
  2. Click the menu item "Open"

File browser controls:

  1. The current directory; click to edit the path
  2. Go up a directory
  3. Go back or forward a directory
  4. Shortcuts, recently used directories, and drives
  5. Sorting options
  6. Search for items in the directory
  7. Show the settings
  8. Click a file to open it

Recent Files

Open the recent files:

  1. Open the "File" menu
  2. Click the menu item "Recent"

Recent files controls:

  1. Click a file to open it
  2. Search for items
  3. Show the settings
