Integrating with Maya

The following steps describe how to setup Maya to use DJV for viewing images and playblasts.

Show the preferences dialog:

  1. Open the "Windows" menu
  2. Click the menu item "Preferences"

Configure the external applications settings:

  1. Go to the applications category
  2. Set the image viewing application, "C:\Program Files\DJV2\bin\djv.exe", and optional flags, "-maya %f"
  3. Set the "Image Sequence" viewing application, "C:\Program Files\DJV2\bin\djv.exe", and optional flags, "-maya %f -r %r -start_end %s %e"
  4. Set the "Audio Video Interleave (AVI) files" viewing application, "C:\Program Files\DJV2\bin\djv.exe", and optional flags, "-maya %f -r %r"
  5. Set the "Quicktime (MOV) files" viewing application, "C:\Program Files\DJV2\bin\djv.exe", and optional flags, "-maya %f -r %r"
